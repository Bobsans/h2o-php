h2o-php by speedmax

A beautiful template markup for PHP in django style...

H2o is template engine for PHP, there are a lot of the concept are inspired by projects like Django templates, Smarty and Jinja, implemented in a way that makes sense in PHP. Author your template the easy way ...

  1. Readable and natural syntax.
  2. Reuse template inclusion and inheritance.
  3. Extensible through filters, tags and template extensions.
  4. Battery included - bundled rich set of filters and tags
  5. Internationalization
  6. Various Caching support with multiple backend

Dependencies

PHP 5.1 +

Install

Download and extract h2o in your project path or your php include path Sample file structure setup 


myawesome_app/
    index.php
    templates/
      index.html
    h2o/
  1. use require 'h2o/h2o.php' in your php statement to include h2o library
  2. Below is a quick start code example to get a kick start
  3. checkout example and specs if you are in the mood for exploration.
templates/index.html 

<body>
    <head><title>{{ page.title }}</title></head>
    <body>
        {{ page.body }}
    </body>
</body>
index.php 

<?php
    require 'h2o/h2o.php';
    # load
    $h2o = new h2o('index.html');
 
    # render your awesome page
    echo $h2o->render(array(
      'title' => 'title of a page',
      'body' => 'Hello world'
    ));
    
    # how about say hello to everyone using h2o() convenience function
    echo h2o('{{ greeting }}')->render(array('greeting'=>'hello every one'));
?>
