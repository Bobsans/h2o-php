    <h3>Features</h3>
    <ol>
      <li>High readability, Natural syntax, Clean separation between logic, presentation and presentation logic</li>
      <li>Template inheritance, Tags, Filters lead you to reusable code</li>      
      <li>Powerful extension architecture, able to extend features on runtime by filters/tags</li>
      <li>Compile h2o template into native PHP code</li>
      <li>H2O has very modular design, you can tailer to need your exact needs.</li>      
      <li>Sandbox template, Handle most template error by H2O engine, server detail won't be exposed, only display data explicitly passed to H2O</li>
    </ol>
    <h3>Performance</h3>
    <ol>
        <li>Lightweight implementation, minimum run time footprint</li>
        <li>Compiled into PHP file with minimum file IO</li>
        <li>Output cache, block level cache</li>
        <li>Designed and Works really well with opt-code cache such as eaccelerator/xcache/APC/Zend platform</li>
    </ol>
